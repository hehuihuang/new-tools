故障排查

如果链接无法打开，请检查：

  1. 确保所有 PDF 工具文件都在 pdf-tools/ 文件夹中
  2. 确保文件名拼写正确（区分大小写）
  3. 如果使用浏览器直接打开文件，某些浏览器可能会阻止跨文件夹访问
  4. 建议使用本地 Web 服务器（如 Live Server）来运行项目
