PDF 工具套件 - 使用说明
================================

问题: 为什么点击链接没有反应？
--------------------------------
这是因为浏览器的安全限制。当你直接双击打开 HTML 文件时，浏览器使用 file:// 协议，
这会阻止跨文件夹的链接访问。

解决方案（选择其中一种）：
================================

方案 1: 使用启动脚本（推荐，最简单）
--------------------------------
1. 双击项目根目录的 "start-server.bat" 文件
2. 等待命令窗口显示 "Serving HTTP on..."
3. 打开浏览器，访问: http://localhost:8000/tooles/index.html
4. 现在所有链接都可以正常工作了！

方案 2: 使用 Python（如果已安装）
--------------------------------
1. 在项目根目录打开命令提示符（cmd）
2. 输入命令: python -m http.server 8000
3. 打开浏览器，访问: http://localhost:8000/tooles/index.html

方案 3: 使用 VS Code（如果你用 VS Code 编辑代码）
--------------------------------
1. 在 VS Code 中打开项目文件夹
2. 安装 "Live Server" 扩展
3. 右键点击 tooles/index.html 文件
4. 选择 "Open with Live Server"

方案 4: 使用诊断工具
--------------------------------
1. 双击打开 "diagnose.html" 文件
2. 按照页面上的说明操作
3. 工具会自动检测问题并提供解决方案

测试链接
================================
打开 "test-links.html" 可以测试所有 PDF 工具的链接是否正常工作。

功能列表
================================
✓ PDF 在线预览 - 查看 PDF 文件，支持翻页和缩放
✓ PDF 合并 - 将多个 PDF 合并为一个
✓ PDF 拆分 - 按页面范围提取 PDF
✓ PDF 重排 - 删除和重新排列 PDF 页面
✓ PDF 水印 - 给 PDF 添加文字水印
✓ PDF 批注 - 在 PDF 上添加批注
✓ PDF 转图片 - 将 PDF 转换为图片格式
✓ PDF 压缩 - 减小 PDF 文件大小

隐私保护
================================
所有 PDF 处理都在您的浏览器本地完成，不会上传到任何服务器。
您的文件完全安全，不会被任何人看到。
